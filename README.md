# AutoSchedule AI

AI-powered WhatsApp scheduling agent that autonomously handles booking, rescheduling and cancellations — no menus, no human intervention. Built with N8N, Google Calendar, Supabase and Redis.

---

## 🚀 Features

- Natural language conversations (no buttons, no menus)
- End-to-end scheduling automation with zero human intervention
- Multi-professional support with individual calendar management
- Duplicate patient detection via email validation
- Intelligent rescheduling — automatically cancels previous bookings before creating new ones
- Persistent session memory powered by Redis
- Google Calendar API integration
- Scalable backend powered by Supabase
- Supervisor + sub-agent architecture for complex conversational workflows

---

## 🧠 Architecture

- Orchestrated via N8N workflows
- LLM-based conversational agent
- Redis for session memory
- Supabase for persistence layer
- Google Calendar API for scheduling
