# AutoSchedule AI 🤖  
### Production-Ready WhatsApp Scheduling Agent powered by AI

AI-powered WhatsApp scheduling system that autonomously handles booking, rescheduling, and cancellations — no menus, no commands, no human intervention.

Built with N8N, LLMs, Google Calendar, Supabase, and Redis.

## 💬 What this agent does

- Schedules, reschedules, and cancels appointments via natural conversation  
- Understands ambiguous inputs like *"tomorrow afternoon"*  
- Prevents double bookings with real-time availability checks  
- Handles multi-professional scheduling seamlessly  
- Maintains conversation memory across sessions  
- Supports scalable, production-ready automation workflows  

---

## 🚀 Features

- Natural language conversations (no buttons, no menus)  
- End-to-end scheduling automation with zero human intervention  
- Multi-professional support with individual calendar management  
- Duplicate patient detection via validation logic  
- Intelligent rescheduling — cancels previous bookings safely  
- Persistent session memory powered by Redis  
- Google Calendar API integration  
- Scalable backend powered by Supabase  
- Supervisor + multi-agent architecture for complex workflows  

---

## 🧠 Architecture

- Orchestrated via N8N workflows  
- Supervisor agent for intent detection and routing  
- Sub-agents for scheduling, knowledge retrieval, and operations  
- Redis for session memory and conversational context  
- Supabase as persistence layer  
- Google Calendar API for scheduling operations  

---

## ⚙️ Tech Stack

- **N8N** — workflow orchestration and automation engine  
- **LLMs (OpenAI / Claude / others)** — natural language understanding  
- **Redis** — session memory and context management  
- **Supabase** — database and scalable backend  
- **Google Calendar API** — scheduling and availability management  
- **WhatsApp API (Evolution API)** — communication layer
